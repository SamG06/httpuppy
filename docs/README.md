# HTTPuppy

![](/logo.png)

A simple abstraction layer for node web servers

## Installation

```cmd
npm i httpuppy
```

```cmd
yarn add httpuppy
```

## Command Line Use

```cmd
npx httpuppy --port=3000 --path=path/to/files
```

spin up a server at `<cwd>/path/to/files`
